# DSA Practice App

Interactive practice tool built in React. Paste into any Claude.ai artifact or deploy standalone.

## Features
- 28 problems across all 14 categories
- AI-powered solution checker (any language)
- Hints, complexity analysis, progress tracking
- Progress persists across sessions via Claude storage API

## Usage
Paste `dsa-practice-app.jsx` into a Claude.ai artifact window and run.
